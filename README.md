wsProxy * Under development
=======

This is a websocket to tcp proxy, written in node.js. It is dynamic and will proxy to various tcp servers.


Installation
----------
```
npm install wsproxy -g
```


Usage
----------

```
wsproxy [-p PORT] 
```
